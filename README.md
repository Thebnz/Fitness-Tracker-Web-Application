# Fitness Tracker Web Application

## Overview

This repository hosts a comprehensive Fitness Tracker Web Application built with HTML, CSS, JavaScript, and Node.js. The application lets users monitor their fitness progress, calculate BMI, read workout blogs for tips and motivation, and manage personal profilesâ€”all via a modern, responsive interface optimized for any device.

## Features

- **Track Fitness Progress:** Log workouts and monitor physical performance over time.
- **BMI Calculator:** Easily calculate and interpret BMI for health awareness.
- **Workout Blogs:** Access curated articles, guides, and motivational content.
- **Profile Management:** Maintain a personalized account with progress stats.
- **Responsive Design:** Enjoy a seamless user experience across desktop, tablet, and mobile.
- **Interactive Tools:** Utilize various interactive elements for engagement and insight.

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **Node.js**

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/habibgurbanovv-a11y/Fitness-tracker-web-application.git
   ```

2. **Install dependencies:**
   ```sh
   cd Fitness-tracker-web-application
   npm install
   ```

3. **Start the application:**
   ```sh
   npm start
   ```

4. **Open in browser:**
   Visit `http://localhost:3000` to view the app.
